# AudioGo

Windows 音訊工具，提供音訊載入、波形顯示、播放與一鍵優化功能。

## 下載

前往右側 [Releases](../../releases) 頁面下載。

## 系統需求

- Windows 7 / 8 / 10 / 11（64-bit）
- 無需安裝，解壓縮後直接執行 `audiogo.exe`

## 使用方式

1. 下載並解壓縮 `AudioGo-win64.zip`
2. 執行資料夾內的 `audiogo.exe`
3. 拖放音訊檔案（WAV、MP3、FLAC）至視窗，或使用選單「檔案 → 開啟」
4. 點選工具列「一鍵優化」進行音訊處理
5. 使用「◉ 原始  ○ 優化」按鈕切換前後對比
6. 在波形上拖曳可框選區段，優化/播放/匯出只針對選取範圍
